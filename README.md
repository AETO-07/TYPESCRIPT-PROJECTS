## Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd TYPE\ SCRIPT\ PROJECTS
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Build the TypeScript files:
   ```bash
   npm run build
   ```

## Running Projects

To run the development server with live reload:
```bash
npm start
```

This will start a local server (using lite-server) and open the index.html page in your browser.
